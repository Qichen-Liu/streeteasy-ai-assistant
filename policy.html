HomeHunt Assistant Privacy Policy v1.0

Effective date: March 14, 2026

HomeHunt Assistant ("the Extension") helps users track listing activity and run AI-assisted evaluations on supported listing pages. This policy explains what data is handled, how it is used, and what controls users have.

1. Data We Handle

  • OpenAI API key (BYOK): Entered by the user in extension settings.
  • Listing activity data: Viewed/contacted/evaluation metadata for tracked listings.
  • Listing page content/context: Listing text/content used only when user triggers AI evaluation.
  • Extension settings: Model choice, report mode, risk priorities, and retention preferences.

2. How We Use Data

  • Track listing status (viewed/contacted) and display it in extension UI.
  • Generate AI evaluation results when the user explicitly clicks AI Evaluate.
  • Persist user preferences and retention settings across sessions.

3. Where Data Is Stored

Extension data is stored locally in the user's browser via chrome.storage.local. This includes settings, API key, and tracked listing metadata.

4. Data Sharing and Third Parties

  • We do not sell user data.
  • We do not share user data with third parties except as needed for AI evaluation requests.
  • When user-triggered AI evaluation is used, relevant listing context and the user-provided API key are sent to OpenAI's API endpoint (https://api.openai.com) to produce evaluation output.

5. User Controls

  • Clear API key from settings at any time.
  • Clear tracked listing data from settings at any time.
  • Adjust retention window (for example 30/90/120/180 days).

6. Security

We use local browser extension storage and do not run remote executable code. Users are responsible for keeping their browser profile and device secure.

7. Children's Privacy

This Extension is not directed to children under 13 and is intended for general real-estate browsing productivity use.

8. Changes to This Policy

We may update this policy from time to time. Material updates will be reflected by updating the effective date.
