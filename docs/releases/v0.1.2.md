# StreetEasy AI Assistant v0.1.2

## Highlights
- Redesigned the **Options / Settings page** with a cleaner, more professional SaaS-style layout.
- Improved visual hierarchy with a centered card layout, refined typography, and tighter spacing.
- Updated action buttons for clearer intent:
  - **Save Settings** now uses a vibrant green primary style.
  - **Clear API Key** and **Clear Tracked Data** now use broader red outlined styles.

## UI/UX Improvements
- Better readability and balance across headers, labels, field text, and helper text.
- More consistent input styling (rounded corners, soft-gray fields, subtle borders).
- Cleaner footer action area with stronger button affordance and visual polish.

## Compatibility / Behavior
- No functional changes to settings behavior.
- Existing controls and workflows remain the same:
  - Save settings
  - Clear API key
  - Clear tracked data
- Existing stored data/settings remain compatible.

## Notes
- This release focuses on **visual polish and usability** of the settings experience.
