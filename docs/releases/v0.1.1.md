# HomeHunt Assistant v0.1.1

## Highlights
- Redesigned in-page HomeHunt Assistant panel UI to match the listing-site style direction.
- Added draggable + collapsible AI panel with bottom-right launcher.
- Added draggable search-results filter widget to hide:
  - viewed listings only
  - viewed + contacted listings
- Added retention controls for tracked listings:
  - 30 / 90 / 120 / 180 days (default 90)
- Added bulk management in Recent Listings:
  - select visible
  - remove selected
- Upgraded extension icon system with unified green-badge icon set.

## Improvements
- Address text in AI panel is selectable/copyable.
- Contact/Viewed status visuals improved with clearer icon states.
- AI risk labels are now human-readable (no underscore-style tokens).

## Bug Fixes
- Fixed cases where viewed tracking could miss listings after browser back/forward navigation.
- Removed duplicate viewed entries in recent lists.
- Fixed select-all mixed state behavior in popup selection UX.
- Fixed filtering behavior that could leave blank gaps or unstable counts on results pages.
- Improved empty-filtered-page handling so page navigation remains usable.
- Fixed blurred assistant header icon by using scalable source for in-page rendering.

## Notes
- This release is focused on UX, tracking reliability, and browsing workflow efficiency.
