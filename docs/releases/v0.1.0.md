# HomeHunt Assistant v0.1.0

## Summary
First MVP release of HomeHunt Assistant for tracking listing activity and running AI-assisted listing evaluations.

## What’s Included
- View tracking on StreetEasy listing pages
- Contacted status toggle and persistence
- Injected listing-side UI card with:
  - Viewed/Contacted chips
  - AI Evaluate action
  - Evaluation summary (price score, quality score, confidence, risks)
- Popup dashboard with:
  - Recent tracked listings
  - Filters (`All`, `Contacted`, `Viewed only`, `Evaluated`)
  - Sorts (`Last viewed`, `Price low-high`, `Price high-low`, `Best AI score`)
  - Remove listing action
- Options page for BYOK configuration:
  - OpenAI API key
  - Model selection
  - Report mode and risk priorities
- Caching + parser hardening for AI responses
- Improved listing detection and route-change handling
- Custom extension icons

## Reliability and UX Improvements Included During v0.1.0
- Deduplication of viewed listings in popup
- Stable fallback listing key logic for non-numeric listing URLs
- Better error handling for missing/invalid API key and rate limits
- Human-readable risk text formatting (no underscore tokens)
- Prompt calibration update for more realistic scoring behavior

## Known Limitations
- BYOK only (no backend proxy mode yet)
- Data stored locally in `chrome.storage.local` only (no account sync)
- AI quality depends on listing page data completeness
- StreetEasy layout changes may require selector updates

## Validation Status
Release-gate checks passed:
- `npm run typecheck`
- `npm run test`
- `npm run build`

Manual verification checklist:
- `docs/e2e-checklist.md`

## Upgrade / Install Notes
1. Build and load `dist/` as unpacked extension in Chrome.
2. Configure OpenAI API key in extension options before using AI Evaluate.
3. Refresh extension after updates in `chrome://extensions`.

## Next Steps (Post v0.1.0)
- Backend-proxy mode (non-BYOK path)
- Sync across browsers/devices
- Enhanced market comp support for pricing evaluation
- Expanded automated tests and integration checks
