# StreetEasy AI Assistant v0.1.3

## Highlights
- Fixed intermittent `Extension context invalidated` errors seen during StreetEasy navigation.
- Improved content script resilience when extension context changes after reload/update.

## Fixes
- Added safe wrappers for `chrome.storage.local.get/set` in content script.
- Added context-invalidation guards to stop observers and cleanly remove extension UI when needed.
- Prevented uncaught promise noise from navigation sync when context is invalidated.

## Stability Impact
- Reduces console/runtime noise in results and listing pages.
- Keeps core workflows stable during extension reload/update cycles.

## Notes
- No user-facing feature changes in this patch.
- Recommended for all users before Chrome Web Store submission/update.
